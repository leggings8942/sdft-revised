# Data Mapping Guide

Map the user's files into these buckets before analysis:

- 来店数データ
- 滞在時間データ
- 回遊データ
- 人気エリアデータ
- 来店者居住地
- 来店者勤務地
- 競合重複状況
- 来店経路
- 来店者行動トピック
- 来店者コホート
- AIカメラ属性・人数・感情


After bucketing:
1. identify what is observed directly
2. identify what needs transformation or aggregation
3. identify what can only be inferred by the model
4. note missing fields that limit confidence
