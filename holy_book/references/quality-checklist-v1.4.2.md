# Quality Checklist v1.4.2

Before finalizing a facility report, check all boxes:

- [ ] Coverage period, granularity, and blind spots are disclosed.
- [ ] Common-area-only vs full-facility coverage is stated.
- [ ] A table exists for Observed / Derived / Assumed / Hypothesis.
- [ ] H-based alert language is proportional to the actual time-series evidence.
- [ ] Demographic segmentation is labeled with confidence.
- [ ] Gauge distortion is marked as exact or proxy.
- [ ] Maxwell parameters have provenance and caveats.
- [ ] Each intervention is tied to evidence.
- [ ] Uncertainty and next data collection steps are present.
