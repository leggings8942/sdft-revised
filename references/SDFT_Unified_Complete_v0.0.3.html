1. 概要・設計思想

v0.0.3 の核心的な変更：𝓣 のベクトル化とハイパーパラメータの廃止

原典（v1.4.2）では 𝓣 を以下のようにスカラーとして定義していた：
$\mathcal{T} = |\log(L_A/L_B)| + \kappa \cdot \Delta\mathrm{Topo} + \lambda \cdot \Delta\mathrm{Op} + \eta \cdot \|hol-I\| + b_{norm} + b_{cog} + \max(0, 1-\alpha_{align})$

問題： κ, λ, η という重み係数は異なる意味・単位を持つ量を合成するための恣意的なパラメータであり、理論的正当化が困難。

v0.0.3 の解決： 𝓣 を5次元ベクトルとして定義することで重み係数を完全に廃止する。 各成分は独立した不整合の次元を表し、「どの次元で不整合が大きいか」が明確になる。
項目v1.4.2 原典v0.0.3
𝓣 の型スカラー（$\mathbb{R}$）5次元ベクトル（$\mathbb{R}^5$）
理論基盤物理学アナロジー（ゲージ理論・位相幾何学・作用素論）情報幾何学・シャノン情報理論
重みパラメータκ, λ, η が必要廃止（ベクトルの各成分がそのまま情報を保持）
社会層$b_{norm}+b_{cog}+\max(0,1-\alpha_{align})$（3項）通信路損失率 $\mathcal{L}(e)$（1項）
集約方法加重和（スカラー）集約しない（5成分を個別保持）
観測可能性ほぼ全項が AssumedT₁・T₂・T₃・T₅ は Derived で計算可能
S/D/H/F の定義変更なし変更なし
位相判定（5相）変更なし変更なし

4. v0.0.3 の 𝓣 定義（5次元ベクトル）

$$\vec{\mathcal{T}}_{IG}(A, B) = \begin{pmatrix} T_1 \\ T_2 \\ T_3 \\ T_4 \\ T_5 \end{pmatrix} = \begin{pmatrix} |\psi_A - \psi_B| \\ d_B(p_A, p_B) \\ D^{(\alpha \neq 0)}(p_A \| p_B) \\ \|\mathrm{Hol}^{(\alpha)}(\gamma) - I\| \\ \mathcal{L}(e) \end{pmatrix} \in \mathbb{R}^5$$
式(v0.0.3-1)：レジーム間テンション — ベクトル定義（ハイパーパラメータなし）
ベクトル設計の原則：
・ハイパーパラメータ（κ, λ, η）を一切持たない
・スカラーに集約しない（5成分を常に個別に保持する）
・各成分は独立した不整合の次元を表す
・T₁〜T₄ は静的な構造差、T₅ はエッジの動的な伝達損失

4-1. $T_1$：対数分配関数差（スケール差）

$$T_1 = |\psi_A - \psi_B|, \qquad \psi(\theta) = \log Z(\theta) = \log \int \exp(\theta \cdot x)\, dx$$
式(v0.0.3-2)

原典との対応

原典の $|\log(L_A/L_B)|$ は「特性長のスケール差」を意図していた。$|\psi_A - \psi_B|$ は指数型分布族のポテンシャル関数の差として同じ「スケール・自由度の乖離」を情報幾何学的に表現する。

観測経路

psi_A = -mean(log p_hat_A(x_t) for x_t in regime_A)  # 負の対数尤度の経験平均
psi_B = -mean(log p_hat_B(x_t) for x_t in regime_B)
T1 = abs(psi_A - psi_B)
証拠分類：Derived（両レジームの時系列データが存在する場合）

4-2. $T_2$：Bhattacharyya距離（状態点の幾何学的ずれ）

$$T_2 = d_B(p_A, p_B) = -\log BC(p_A, p_B), \qquad BC(p_A, p_B) = \int \sqrt{p_A(x)\, p_B(x)}\, dx$$
式(v0.0.3-3)

Fisher-Rao 測地距離との関係

$$d_{FR}(p_A, p_B) \approx 2\sqrt{d_B(p_A, p_B)}$$
式(v0.0.3-4)：Bhattacharyya は Fisher-Rao の単調近似

原典との対応

原典の $\Delta\mathrm{Topo}$ は「状態点の幾何学的ずれ」を意図していた（実装はパワースペクトルの JSD だった）。$d_B$ は統計多様体上の2点間の幾何学的距離の近似として、原典の意図に最も忠実な実現。

T₃との独立性

$T_2$ は対称（$BC = \int\sqrt{p_A p_B}$ を使う）。$T_3$ は非対称（α≠0）。 α≠0 を守る限り T₂ と T₃ は独立した情報を持つ。 α=0 のとき T₃ は JSD となり BC と等価になるため α=0 は禁止。

観測経路

p_A, p_B = empirical_distribution_shared_bins(data_A, data_B, bins=50)
BC = sum(sqrt(p_A[i] * p_B[i]) for i in range(bins))
T2 = -log(max(BC, 1e-9))
証拠分類：Derived（経験分布から直接計算・分布仮定不要・双峰対応）

4-3. $T_3$：α-ダイバージェンス（非対称な意味的差・α≠0）

$$T_3 = D^{(\alpha)}(p_A \| p_B) = \frac{4}{1-\alpha^2}\left(1 - \int p_A(x)^{\frac{1-\alpha}{2}} p_B(x)^{\frac{1+\alpha}{2}}\, dx\right), \quad \alpha \neq 0$$
式(v0.0.3-5)
α形式意味用途
$\alpha = +1$$D_{KL}(p_A \| p_B)$A の視点から B の「意外さ」A が送り手・基準の場合
$\alpha = -1$$D_{KL}(p_B \| p_A)$B の視点から A の「意外さ」B が送り手・基準の場合
$\alpha = 0$（禁止）JSDBC を使う → T₂ と重複使用禁止

原典との対応

原典の $\Delta\mathrm{Op}$（演算子差）は「意味変換のズレ」を意図していた。α-ダイバージェンスは「どちらのレジームの視点から差を測るか」を α で制御できる非対称な距離として、「意味変換の方向性」という元の意図に対応する。

観測経路

# α=1（KL ダイバージェンス）の場合
T3 = sum(p_A[i] * log(p_A[i] / p_B[i]) for i in range(bins))

# α=-1（逆 KL）の場合
T3 = sum(p_B[i] * log(p_B[i] / p_A[i]) for i in range(bins))

証拠分類：Derived（経験分布から計算可能）
⚠️ 使用した α 値をレポートに必ず明示すること

4-4. $T_4$：統計的ホロノミー（多様体の曲率差）

$$T_4 = \|\mathrm{Hol}^{(\alpha)}(\gamma) - I\|, \qquad \mathrm{Hol}^{(\alpha)}(\gamma) = \mathcal{P}\exp\!\left(\oint_\gamma \Gamma^{(\alpha)}_{ijk}\, d\theta^k\right)$$
式(v0.0.3-6)
$$R_{ijkl} = \frac{1-\alpha^2}{4}\left(g_{il}g_{jk} - g_{ik}g_{jl}\right)$$
式(v0.0.3-7)：指数型分布族の曲率テンソル（閉形式）

原典との対応

原典の $\|hol(W_{cycle})-I\|$ と数学的に最も対応する項。原典は四元数（観測経路なし）だが、統計的ホロノミーは Fisher 情報行列から近似推定できる。

観測経路（近似）

# 正規分布近似による簡易推定
G_A = 1 / var(regime_A)           # スカラーFisher情報（正規分布近似）
G_B = 1 / var(regime_B)
R_A = (1 - alpha**2) / 4 * G_A**2 # 曲率テンソル近似
R_B = (1 - alpha**2) / 4 * G_B**2
T4  = abs(R_A - R_B) / max(R_A, R_B)  # [0,1] に正規化

証拠分類：Assumed（近似推定）

4-5. $T_5$：通信路損失率（旧社会層の統合）

$$T_5 = \mathcal{L}(e) = 1 - \frac{I(X_A;\, X_B)}{H(X_A)}, \qquad I(X_A; X_B) = H(X_A) + H(X_B) - H(X_A, X_B)$$
式(v0.0.3-8)
$\mathcal{L}$ の値意味
$\mathcal{L} = 0$完全に伝わる（損失ゼロ）
$0 < \mathcal{L} < 1$部分的に損失（現実の多くのケース）
$\mathcal{L} = 1$全く伝わらない（送受信が独立）

原典の社会層との対応

旧変数本質通信路損失における役割
$b_{norm}$通過の許容度チャネルが制度的に閉じている → $I$ が下がる
$b_{cog}$意味の保存率ノイズによる意味の劣化 → $H(X_A,X_B)$ が上がる
$\max(0,1-\alpha_{align})$方向の一致度符号化・復号化のミスマッチ → $I$ が下がる

観測経路

H_A  = calc_entropy(regime_A_data)
H_B  = calc_entropy(regime_B_data)
H_AB = calc_joint_entropy(regime_A_data, regime_B_data)
I_AB = H_A + H_B - H_AB
T5   = max(0.0, min(1.0, 1.0 - I_AB / (H_A + 1e-9)))

証拠分類：Derived（同時観測データがある場合）
          Assumed（推定・片側のみの場合）
