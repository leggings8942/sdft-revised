第1章　概要と設計思想

1.1 SDFT とは何か

SDFT（Structural Dynamics Field Theory）は、複合系（マーケット・組織・施設・都市・複合施設など）の構造を 情報幾何学 と 統計力学 のアナロジーで記述する分析フレームワークである。

目的は以下の3つに集約される：

  1. 構造の診断：系が「健全な動的平衡」「硬直」「過成長」「崩壊予兆」のどの状態にあるかを判定する
  2. 状態の予測：放置した場合に系がどう変化するかを予測する
  3. 介入の設計：最小のインパクトで望ましい状態を維持・回復するための施策を設計する

1.3 7つの設計原則

原則1：情報幾何学・熱力学的アナロジーで定義する
全ての変数を Fisher 情報行列・Bhattacharyya 距離・α-ダイバージェンス・自由エネルギー・グランドポテンシャルなど、確立された数学的体系から導出する。
原則2：全変数に証拠分類ラベルを付与する
Observed（実測）/ Derived（導出）/ Assumed（仮定）/ Hypothesis（推論）の4層で全変数を分類する。
原則3：𝓣 はハイパーパラメータなしの5次元ベクトル
レジーム間テンション 𝓣 を構成する 5 つの成分 (T₁, T₂, T₃, T₄, T₅) は、それぞれ独立した情報幾何学的・統計的距離として定義され、外部から設定するハイパーパラメータを持たない。
原則4：𝓣 をスカラーに集約しない
𝓣 の 5 成分は独立した不整合の次元を表すため、重み付き和などでスカラーに集約することは禁止される。
原則5：Φ は放置すれば減少する
グランドポテンシャル Φ = F - μN の全微分 dΦ = -S·dT - P·dV - N·dμ が、外部介入のない散逸過程で常に負になるように定義する。
原則6：介入は最小インパクトで Φ ≥ Φ₀ を維持する
介入を「min Σ‖u_k‖² + λK s.t. Φ(t) ≥ Φ₀」という最適化問題として定式化する。
原則7：学術的に確立していない概念を恣意的に持ち込まない
アフォーダンス・ゲージ場・Maxwell 方程式・Hamiltonian 力学などの概念は、SDFT の文脈で「確立された計算手法」を持たない限り、SDFT に組み込まない。

第2章　Layer 1：構造場変数

2.1 S：熱力学的エントロピー Derived

定義

Hbit = -Σi pi log2(pi)   [bit]
S = kB · ln(2) · Hbit   [J/K]
kB = 1.380649 × 10-23 J/K（ボルツマン定数）

解釈

S の状態意味
S 高分布が広い・多様 → 無秩序・散逸が進んでいる
S 低分布が狭い・集中 → 秩序・安定している

理論的根拠

シャノン情報理論におけるエントロピー（bit 単位）を、ボルツマン定数 kB と ln(2) を介して熱力学的エントロピー（J/K 単位）に変換する。これは Jaynes（1957）の最大エントロピー原理により、情報論的エントロピーと熱力学的エントロピーが数学的に同型であることに基づく。

2.2 D：フラクタル次元 Derived

定義（ヒグチ法）

D = limk→0 [log L(k) / log(1/k)]   ∈ [1.0, 2.0]

解釈

D の値意味
D → 1.0単純な曲線（規則的・低複雑性）
D → 2.0完全ランダム（不規則・高複雑性）

S が「分布の広がり」を測るのに対し、D は「時間的パターンの複雑さ」を測る。

2.3 H：Hurst 指数 Derived

定義（ラグ別標準偏差法）

std(xt+s - xt) ~ sH
H = slope of log(std) vs log(s)   ∈ [0.0, 1.0]

解釈

H の値意味
H > 0.5持続性（トレンドが続く傾向）
H = 0.5ランダムウォーク
H < 0.5反持続性（反転しやすい）

重要性

H は相転移の最も重要な先行指標である。崩壊の前に H が低下する性質を持つため、変化予兆検出において中核的な役割を果たす。

⚠️ 短い系列（n < 200）では推定精度が低い。

第3章　Layer 2：Fisher 情報行列とその導出量

3.1 Fisher 情報行列 G Derived

定義

Gij(θ) = E[ (∂log p / ∂θi)(∂log p / ∂θj) ]

正規分布 𝒩(μ, σ²) の場合（閉形式）：

G = [[ 1/σ², 0 ],
     [ 0, 2/σ⁴ ]]

パラメータは θ = (μ, σ²)。

データの標準化

データを 標準化してから G を計算する（平均0・分散1にスケーリング）。標準化しないとデータのスケールに依存して T, U, F が天文学的な値になるため。

標準化後は σ² ≈ 1 となり、G ≈ diag(1, 2) となる。

解釈

G は「統計多様体上の計量テンソル」であり、パラメータ感度の幾何を与える。

G の状態意味
G 大パラメータへの感度が高い → エネルギーランドスケープが急峻
G 小パラメータへの感度が低い → 散逸が進んでいる

3.2 T：情報的温度 Derived

定義

T = tr(G⁻¹(θ)) / (d · kB)   [K]

理論的根拠（Jaynes 1957）

熱平衡分布（ボルツマン）と最大エントロピー分布（指数型分布族）の数学的同型性から、厳密な対応が導かれる：

kB T ↔ tr(G⁻¹) / d

これは「系の揺らぎのエネルギースケール」を表す。

3.3 U：内部エネルギー Derived

定義

U = tr(G(θ)) / d   [J相当]

理論的根拠

等分配則により各自由度に kBT/2 のエネルギーが分配される。Fisher 行列のトレースはその「情報的エネルギー密度」に対応する。

3.4 F：自由エネルギー Derived

定義

F = U - T · S   [J]

展開すると：

F = tr(G)/d - ln(2) · Hbit · tr(G⁻¹)/d

単位の整合性

T · S の計算で kB がキャンセルされて単位 [J] が得られる：

T · S = [tr(G⁻¹)/(d·kB)] · [kB·ln2·Hbit] = ln2 · Hbit · tr(G⁻¹) / d

解釈

F の状態意味
F > 0系が外部に仕事できる状態（エネルギー的に豊か）
F < 0外部から仕事を与えないと維持できない状態（散逸進行）

3.5 V：統計多様体の体積 Derived

定義

V = ∫Ω √det(G(θ)) dθ   [m³相当]

実用的な近似実装：V ≈ √det(G) · θranged

理論的根拠

√det(G) は Jeffreys 事前分布密度であり、パラメータ変換に対して不変な事前分布の密度を表す。

第4章　Layer 2：レジーム間テンション 𝓣

4.1 設計原則

𝓣(A, B) = (T₁, T₂, T₃, T₄, T₅) ∈ ℝ⁵
原則内容
1ハイパーパラメータ（重み係数）を持たない
2スカラーに集約しない
3各成分は独立した不整合の次元
4加算・重み付け合成を行わない

4.2 T₁：対数分配関数差（スケール差）Derived

定義

T₁ = |ψA - ψB|,   ψ(θ) = log Z(θ)

経験的推定：ψ ≈ -E[log p(x)]（負の対数尤度の経験平均）

意味

2つのレジームの状態空間のスケール・自由度の乖離。

4.3 T₂：Bhattacharyya 距離（状態点の幾何学的ずれ）Derived

定義

T₂ = dB(pA, pB) = -log ∫√(pA(x) · pB(x)) dx = -log BC(pA, pB)

Fisher-Rao 測地距離との関係

dFR(pA, pB) ≈ 2√T₂   （単調近似）

これは化学ポテンシャル μ = 2√T₂ に直結する。

T₃ との独立性

  • T₂ は対称（BC ベース）
  • T₃ は非対称（α ≠ 0）
  • α ≠ 0 を守る限り T₂ と T₃ は独立した情報を持つ

4.4 T₃：α-ダイバージェンス（非対称な意味的差）Derived

定義

T₃ = D(α)(pA‖pB) = (4/(1-α²)) · (1 - ∫ pA(1-α)/2 · pB(1+α)/2 dx)
⚠️ α = 0 は禁止（T₂ の Bhattacharyya と重複するため）。α = ±1 を使用する。
α形式意味
+1DKL(pA‖pB)A の視点から見た B の「意外さ」
-1DKL(pB‖pA)B の視点から見た A の「意外さ」

4.5 T₄：統計的ホロノミー（多様体の曲率差）Assumed

定義

T₄ = ‖Hol(α)(γ) - I‖

実用的な近似実装：非標準化分散の対数行列式差

T₄ ≈ |log det GA - log det GB| / max(|log det GA|, |log det GB|)

ここで det G = 2/σ⁶（非標準化）。

意味

複数レジームを巡る閉ループで情報幾何学的な整合性が保たれるか。多様体の曲率差を反映する。

4.6 T₅：通信路損失率 Derived

定義

T₅ = ℒ(e) = 1 - I(XA; XB) / H(XA)
I(XA; XB) = H(XA) + H(XB) - H(XA, XB)
ℒ意味
0完全に伝わる（損失ゼロ）
1全く伝わらない

意味

エッジ通過時の情報損失率。Shannon（1948）の通信路符号化定理に基づく。

第5章　Layer 3：グランドポテンシャル Φ

5.1 定義

Φ = F - μN   [J]

ここで：

  • F = U - T·S（自由エネルギー）
  • μ = 2√T₂（化学ポテンシャル＝1人の移動コスト）[J/人]
  • N = 移動する人数 [人] Observed

展開すると：

Φ = tr(G)/d - ln(2) · Hbit · tr(G⁻¹)/d - 2√T₂ · N

5.2 全微分

dΦ = -S·dT - P·dV - N·dμ

Ttr(G⁻¹)/(d·kB)KDerived
Psign(dΦ)·‖dΦ‖GJDerived

5.3 放置時に Φ が減少することの確認

項放置時の変化符号Φ への影響
-S·dT散逸 → T 上昇（dT > 0）-(+)(+) < 0✅ 減少

3項全てが減少方向に作用するため、外部介入がない限り Φ は単調減少する。

第8章　介入最適化（§3）

8.1 最適化問題の定式化

min{uk}, K   Σk=1K ‖uk‖² + λK
s.t.   Φ(t) ≥ Φ₀   ∀t

「最小のインパクト・最小の介入回数で Φ(t) ≥ Φ₀ を維持・増加させる」

記号意味
uk第 k 回目の介入操作量
K総介入回数
λ介入回数ペナルティ係数
Φ₀介入閾値（維持すべき下限）

8.2 操作変数と感度

操作変数∂Φ/∂ui介入の実体
u₁ = N（人数）-μ = -2√T₂集客・送客・人員配置・流入制御
u₂ = T₂（距離）-N/√T₂レジーム間の体験・文化・情報の差を縮める

8.3 アルゴリズム（9ステップ）

Step 1: Φ(t) を計算
Step 2: dΦ/dt ≈ (Φ(t) - Φ(t-Δt))/Δt を推定
Step 3: Φ̂(t+τ) = Φ(t) + dΦ/dt·τ を予測
Step 4: Φ̂(t+τ) < Φ₀ なら介入、そうでなければスキップ
Step 5: ΔΦ_need = Φ₀ - Φ̂(t+τ) を計算
Step 6: priority_i = |∂Φ/∂u_i| / c_i を計算
Step 7: best u* = argmax_i priority_i を選択
Step 8: 最小インパクト Δu* = ΔΦ_need / (∂Φ/∂u*) を計算
Step 9: 介入実行 → Φ_after = Φ_current + (∂Φ/∂u*) · Δu*

8.4 λ のトレードオフ

λ挙動
λ 大介入回数を減らすことを優先（1回あたりのインパクトが大きい）
λ 小インパクトを小さくすることを優先（介入回数が増える）

エルボー法で最適 λ を自動選択する。

8.5 Φ₀ の自動設定

Living System 相の境界条件から自動導出する：

Fboundary = U - T · Starget,   Starget ≈ 0.55 · kB
Φ₀ = Fboundary - μ · N

8.6 τ の自動設定

H の範囲τ理由
H ≥ 0.65 ステップ持続傾向が強く長期予測が有効
H ≥ 0.453 ステップ中程度の持続性
H ≥ 0.32 ステップ持続性が低い
H < 0.31 ステップ反持続傾向・最短で予測

12.1 基本動作原則

  3. 𝓣 をスカラー化しない（5成分を個別に保持・表示）
