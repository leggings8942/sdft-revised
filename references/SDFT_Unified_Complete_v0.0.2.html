式(OLD-1)：v1.4.2 レジーム間テンション定義式

4. 新定義：𝓣 の情報幾何学的再定義（v0.0.2）

完全な定義式

$$\mathcal{T}_{IG}(A, B) = \underbrace{|\psi_A - \psi_B|}_{T_1} + \underbrace{\kappa \cdot \|\mathrm{spec}(G_A) - \mathrm{spec}(G_B)\|}_{T_2} + \underbrace{\lambda \cdot D^{(\alpha)}(p_A \| p_B)}_{T_3} + \underbrace{\eta \cdot \|\mathrm{Hol}^{(\alpha)}(\gamma) - I\|}_{T_4} + \underbrace{\mu \cdot \mathcal{L}(e)}_{T_5}$$
式(v0.0.2-1)：レジーム間テンション — 情報幾何学・情報理論統合定義

各項が測るものをレイヤー別に整理すると：

2つの層からなる構造：
T₁〜T₄：静的構造差（レジーム A と B の「場」としての差）
T₅：動的伝達損失（エッジ $e$ を通過する際の「流れ」の損失）

全項が情報量次元（または情報量次元に正規化可能な無次元量）で統一されている。

4-1. $T_1$：対数分配関数差

$$T_1 = |\psi_A(\theta_A) - \psi_B(\theta_B)|$$
式(v0.0.2-2)

理論的背景

指数型分布族 $p(x; \theta) = \exp(\theta \cdot x - \psi(\theta))$ において、 対数分配関数（ポテンシャル関数）$\psi(\theta) = \log Z(\theta)$ は 分布が「どれだけの状態空間を有効に使っているか」を表す。 情報幾何学では $\psi(\theta)$ は統計多様体上のポテンシャルであり、 その差は2つのレジームの「自由度・スケールの乖離」を測る。

旧定義 $\left|\log\frac{L_A}{L_B}\right|$ との対応

旧定義の特性長比 $L_A / L_B$ は「スケールの比」を測ることを意図していた。 $\psi_A - \psi_B$ は分配関数の対数差であり、同じく「スケール・自由度の対数差」として機能する。 ただし観測からの計算経路が明確に存在する点が異なる。

観測からの計算経路

# 経験的な対数分配関数の推定（負の対数尤度から）
psi_A ≈ -mean(log p_hat_A(x_t) for x_t in regime_A)
psi_B ≈ -mean(log p_hat_B(x_t) for x_t in regime_B)
T1 = abs(psi_A - psi_B)

証拠分類：Derived（各レジームの時系列データが存在する場合）

4-2. $T_2$：Fisher情報行列スペクトル差

$$T_2 = \kappa \cdot \|\mathrm{spec}(G_A) - \mathrm{spec}(G_B)\|$$
式(v0.0.2-3)

ここで Fisher 情報行列 $G$ は：

$$G_{ij}(\theta) = \mathbb{E}\!\left[\frac{\partial \log p(x;\theta)}{\partial \theta_i}\frac{\partial \log p(x;\theta)}{\partial \theta_j}\right]$$
式(v0.0.2-4)：Fisher情報行列

理論的背景

Fisher 情報行列は統計多様体上の計量テンソル（Riemannian metric）であり、 その固有値スペクトルは「その分布が持つ変化の方向性と感度の構造」を表す。 $\mathrm{spec}(G)$ のズレは、2つのレジームが「どれだけ異なる統計的感度構造を持つか」、 すなわち 多様体の形（曲率構造）の違い に対応する。

旧定義 $\kappa \cdot \Delta\mathrm{Topo}$ との対応

旧定義の「トポロジー差」は位相幾何学的な構造の差を意図していた。 Fisher スペクトル差は統計多様体上の「幾何学的構造の差」であり、 情報幾何学の文脈でのトポロジー差に対応する。

観測からの計算経路

# スコア関数の経験的共分散行列として推定
score_A = [grad_log_p(x_t) for x_t in regime_A]
G_hat_A = (1/n) * sum(s @ s.T for s in score_A)
G_hat_B = (1/n) * sum(s @ s.T for s in regime_B)

eigvals_A = sorted(eigenvalues(G_hat_A))
eigvals_B = sorted(eigenvalues(G_hat_B))
T2 = kappa * norm(eigvals_A - eigvals_B)

証拠分類：Assumed（スコア関数の推定に仮定が必要）

4-3. $T_3$：$\alpha$-ダイバージェンス

$$T_3 = \lambda \cdot D^{(\alpha)}(p_A \| p_B)$$
式(v0.0.2-5)

ここで $\alpha$-ダイバージェンスは：

$$D^{(\alpha)}(p_A \| p_B) = \frac{4}{1-\alpha^2}\left(1 - \int p_A(x)^{\frac{1-\alpha}{2}}\, p_B(x)^{\frac{1+\alpha}{2}}\, dx\right), \quad \alpha \neq \pm 1$$
式(v0.0.2-6)：α-ダイバージェンス

特殊ケース

$\alpha$形式意味推奨用途
$\alpha = 1$$D_{KL}(p_A \| p_B)$A の視点から B を見たズレA が基準（送り手）の場合
$\alpha = -1$$D_{KL}(p_B \| p_A)$B の視点から A を見たズレB が基準（受け手）の場合
$\alpha = 0$Jensen-Shannon 距離対称・どちらの視点でも同じ対称な分析（推奨デフォルト）

理論的背景

$\alpha$-ダイバージェンスは情報幾何学の中心的な概念であり、 $\alpha$-接続の下での「分布間の情報的距離」を与える。 KL ダイバージェンス（$\alpha = \pm 1$）はその特殊ケース。 パラメータ $\alpha$ を制御することで、どちらのレジームの視点から差を測るかを調整できる。

旧定義 $\lambda \cdot \Delta\mathrm{Op}$ との対応

旧定義の「演算子差」は2つのレジーム間の「変換の差」を意図していた。 $D^{(\alpha)}(p_A \| p_B)$ は状態分布そのものの差を情報量として測り、 観測データから直接計算できる最も強い証拠根拠を持つ項である。

観測からの計算経路

# 経験分布から直接計算（最も観測根拠が強い）
p_hat_A = empirical_distribution(regime_A_data)
p_hat_B = empirical_distribution(regime_B_data)

# α=0（Jensen-Shannon）を推奨デフォルトとする
p_M = 0.5 * p_hat_A + 0.5 * p_hat_B
JSD = 0.5 * KL(p_hat_A, p_M) + 0.5 * KL(p_hat_B, p_M)
T3 = lambda_ * sqrt(JSD)  # JSD は距離として使う場合は平方根

証拠分類：Derived（両レジームの時系列データが存在する場合）

4-4. $T_4$：統計的ホロノミー

$$T_4 = \eta \cdot \|\mathrm{Hol}^{(\alpha)}(\gamma) - I\|$$
式(v0.0.2-7)

ここで $\alpha$-接続のホロノミーは：

$$\mathrm{Hol}^{(\alpha)}(\gamma) = \mathcal{P}\exp\!\left(\oint_\gamma \Gamma^{(\alpha)}_{ijk}\, d\theta^k\right)$$
式(v0.0.2-8)：α-接続のホロノミー（経路順序積）

Christoffel 記号は：

$$\Gamma^{(\alpha)}_{ijk} = \mathbb{E}\!\left[\partial_i \partial_j \log p \cdot \partial_k \log p\right] + \frac{1-\alpha}{2}\,\mathbb{E}\!\left[\partial_i \log p \cdot \partial_j \log p \cdot \partial_k \log p\right]$$
式(v0.0.2-9)：α-接続の Christoffel 記号

理論的背景

統計多様体上で閉ループ $\gamma$（複数のレジームを巡る経路）を一周したとき、 $\alpha$-接続のホロノミーが単位行列 $I$ からズレる量は その多様体の曲率（Riemann 曲率テンソル）に対応する。 これは「複数のレジームをまたいで一周したとき、情報幾何学的に元に戻れるか」を測る量であり、 元の四元数ホロノミーと 構造的に同型 である。

$$R_{ijkl} = \frac{1-\alpha^2}{4}\left(g_{il}g_{jk} - g_{ik}g_{jl}\right)$$
式(v0.0.2-10)：統計多様体の曲率テンソル（指数型分布族）

旧定義 $\eta \cdot \|hol(W_{cycle}) - I\|$ との対応

旧定義の四元数ホロノミー $\|hol(W_{cycle}) - I\|$ と数学的に最も対応する項。 ただし旧定義では $W_{ij}$（四元数）の観測経路がなかったのに対し、 統計的ホロノミーは Fisher 情報行列から近似推定できる。

観測からの計算経路

# Fisher行列の曲率テンソルから近似推定
# 実用上は以下の近似を使う（指数型分布族での閉形式）
R_approx = (1 - alpha**2) / 4 * (g_il*g_jk - g_ik*g_jl)
Hol_approx ≈ I + integral(R_approx over loop area)
T4 = eta * norm(Hol_approx - I)

証拠分類：Assumed（近似推定のため）

4-5. $T_5$：通信路損失率（旧社会層の統合）

$$T_5 = \mu \cdot \mathcal{L}(e), \qquad \mathcal{L}(e) = 1 - \frac{I(X_A;\, X_B)}{H(X_A)}$$
式(v0.0.2-11)：通信路損失率

ここで：

$$I(X_A; X_B) = H(X_A) + H(X_B) - H(X_A, X_B)$$
式(v0.0.2-12)：相互情報量

理論的背景

シャノンの通信路符号化定理において、通信路容量は相互情報量 $I(X;Y)$ で与えられる。 $\mathcal{L}(e) = 1 - I(X_A; X_B)/H(X_A)$ は「送り手の情報のうち受け手に届かなかった割合」 すなわち 情報損失率 を表す。

$\mathcal{L}(e)$ の値意味
$\mathcal{L} = 0$完全に伝わる（チャネル容量 = 送り手のエントロピー）
$0 < \mathcal{L} < 1$部分的に損失（現実の多くのケース）
$\mathcal{L} = 1$全く伝わらない（送り手と受け手が独立）

旧社会層 $b_{norm} + b_{cog} + \max(0, 1-\alpha_{align})$ との対応

旧変数本質通信路損失における役割
$b_{norm}$（規範的障壁） 通過の許容度 チャネルが制度的に閉じている → $I(X_A;X_B)$ が下がる
$b_{cog}$（認知的障壁） 意味の保存率 ノイズによる意味の劣化 → $H(X_A, X_B)$ が上がる
$\max(0, 1-\alpha_{align})$（方向不一致） 意図と行動可能性の一致 符号化・復号化のミスマッチ → $I(X_A;X_B)$ が下がる

観測からの計算経路

# 経験的な相互情報量推定
H_A  = calc_entropy(regime_A_data)
H_B  = calc_entropy(regime_B_data)
H_AB = calc_joint_entropy(regime_A_data, regime_B_data)
I_AB = H_A + H_B - H_AB

# 損失率
L_e = 1.0 - (I_AB / (H_A + eps))
L_e = max(0.0, min(1.0, L_e))  # [0,1] にクリップ
T5  = mu * L_e

証拠分類：Derived（両レジームのデータが存在する場合）/ Assumed（片側のみの場合）

9. 用語集

用語定義
統計多様体確率分布の集合に幾何学的構造を与えた空間。$\mathcal{M} = \{p(x;\theta) \mid \theta \in \Theta\}$
Fisher情報行列統計多様体上の計量テンソル。$G_{ij} = \mathbb{E}[\partial_i \log p \cdot \partial_j \log p]$
対数分配関数指数型分布族の正規化定数の対数。$\psi(\theta) = \log \int \exp(\theta \cdot x) dx$
α-ダイバージェンス$\alpha$-接続の下での分布間の情報的距離。KL ダイバージェンスの一般化。
