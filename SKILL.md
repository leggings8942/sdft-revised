## 𝓣（レジーム間テンション）の定義

### 原典（v1.4.2）との対比

| 項 | 原典 v1.4.2 の定義 | v0.0.3 の定義 | 理論基盤の変化 |
|----|-------------------|--------------|-----------| 
| $T_1$ | $\left\|\log\frac{L_A}{L_B}\right\|$ | $\|\psi_A - \psi_B\|$ | ゲージ特性長 → **対数分配関数差** |
| $T_2$ | $\kappa \cdot \Delta\mathrm{Topo}$ | $d_B(p_A, p_B)$ | 位相幾何学 → **Bhattacharyya距離** |
| $T_3$ | $\lambda \cdot \Delta\mathrm{Op}$ | $D^{(\alpha \neq 0)}(p_A\|p_B)$ | 作用素論 → **α-ダイバージェンス（α≠0）** |
| $T_4$ | $\eta \cdot \|hol(W_{cycle})-I\|$ | $\|\mathrm{Hol}^{(\alpha)}(\gamma)-I\|$ | 四元数ゲージ → **統計的ホロノミー** |
| $T_5$ | $b_{norm}+b_{cog}+\max(0,1-\alpha_{align})$ | $\mathcal{L}(e)$ | 社会学・認知科学 → **通信路損失率** |

原典では各項にハイパーパラメータ（κ, λ, η）を掛けて加算していた。
v0.0.3 ではベクトル化によりハイパーパラメータを廃止し、
各成分をそのまま独立した次元として保持する。

### 各成分の定義

#### $T_1$：対数分配関数差（スケール差）

$$T_1 = |\psi_A - \psi_B|$$

- **意味：** 2つのレジームの状態空間のスケール・自由度の乖離
- **対応する原典の概念：** レジーム特性長比 $|log(L_A/L_B)|$
- **観測経路：** 各レジームの時系列データから対数尤度を推定

```python
psi_A = -mean(log p_hat_A(x_t) for x_t in regime_A)
psi_B = -mean(log p_hat_B(x_t) for x_t in regime_B)
T1 = abs(psi_A - psi_B)
```

証拠分類：**Derived**（時系列データが存在する場合）

---

#### $T_2$：Bhattacharyya距離（状態点の幾何学的ずれ）

$$T_2 = d_B(p_A, p_B) = -\log BC(p_A, p_B) = -\log \int \sqrt{p_A(x)\, p_B(x)}\, dx$$

- **意味：** 統計多様体上の2点間の幾何学的距離（Fisher-Rao測地距離の近似）
- **対応する原典の概念：** トポロジー差 $\Delta\mathrm{Topo}$（「状態点の幾何学的ずれ」）
- **Fisher-Raoとの関係：** $d_{FR}(p_A, p_B) \approx 2\sqrt{d_B(p_A, p_B)}$（単調近似）
- **T₃との独立性：** $T_2$ は対称（BCベース）、$T_3$ は非対称（α≠0）→ 独立

```python
# 経験分布から直接計算（分布仮定不要・双峰データにも対応）
BC = sum(sqrt(p_A[i] * p_B[i]) for i in bins)
T2 = -log(BC)
```

証拠分類：**Derived**（経験分布から計算可能）

---

#### $T_3$：α-ダイバージェンス（非対称な意味的差）

$$T_3 = D^{(\alpha)}(p_A \| p_B) = \frac{4}{1-\alpha^2}\left(1 - \int p_A^{\frac{1-\alpha}{2}} p_B^{\frac{1+\alpha}{2}}\, dx\right), \quad \alpha \neq 0$$

- **意味：** 意味変換の非対称な差。「どちらの視点から見た差か」を α で制御
- **対応する原典の概念：** 演算子差 $\Delta\mathrm{Op}$（「意味変換のズレ」）
- **⚠️ α = 0 は禁止：** $\alpha=0$ は $T_2$ と重複する（ともに BC を使うため）

| α | 形式 | 意味 |
|---|------|------|
| $\alpha = +1$ | $D_{KL}(p_A \| p_B)$ | A の視点から B の「意外さ」 |
| $\alpha = -1$ | $D_{KL}(p_B \| p_A)$ | B の視点から A の「意外さ」 |

```python
# α=1（KL）の場合
T3 = sum(p_A[i] * log(p_A[i] / p_B[i]) for i in bins)

# α=-1（逆KL）の場合
T3 = sum(p_B[i] * log(p_B[i] / p_A[i]) for i in bins)
```

証拠分類：**Derived**（経験分布から計算可能）

---

#### $T_4$：統計的ホロノミー（多様体の曲率差）

$$T_4 = \|\mathrm{Hol}^{(\alpha)}(\gamma) - I\|$$

$$\mathrm{Hol}^{(\alpha)}(\gamma) = \mathcal{P}\exp\!\left(\oint_\gamma \Gamma^{(\alpha)}_{ijk}\, d\theta^k\right)$$

- **意味：** 複数のレジームを巡る閉ループで情報幾何学的な整合性が保たれるか
- **対応する原典の概念：** 四元数ホロノミー $\|hol(W_{cycle})-I\|$（構造的に同型）
- **差異：** 原典は四元数（観測経路なし）、v0.0.3 は Fisher行列から推定可能

```python
# 正規分布近似による簡易推定
G_A = 1 / var(regime_A)  # スカラーFisher情報（正規分布）
G_B = 1 / var(regime_B)
R_A = (1 - alpha**2) / 4 * G_A**2  # 曲率テンソル近似
R_B = (1 - alpha**2) / 4 * G_B**2
T4 = abs(R_A - R_B) / max(R_A, R_B)
```

証拠分類：**Assumed**（近似推定）

---

#### $T_5$：通信路損失率（伝達損失）

$$T_5 = \mathcal{L}(e) = 1 - \frac{I(X_A;\, X_B)}{H(X_A)}$$

$$I(X_A; X_B) = H(X_A) + H(X_B) - H(X_A, X_B)$$

- **意味：** エッジ通過時の情報損失率。$\mathcal{L}=0$ で完全伝達、$\mathcal{L}=1$ で全損
- **対応する原典の概念：** 社会層 $b_{norm}+b_{cog}+\max(0,1-\alpha_{align})$

| 旧変数 | 通信路損失における役割 |
|--------|---------------------|
| $b_{norm}$（規範的障壁） | チャネルが制度的に閉じている |
| $b_{cog}$（認知的障壁） | ノイズによる意味の劣化 |
| $\alpha_{align}$（方向不一致） | 符号化・復号化のミスマッチ |

```python
H_A  = calc_entropy(regime_A_data)
H_B  = calc_entropy(regime_B_data)
H_AB = calc_joint_entropy(regime_A_data, regime_B_data)
I_AB = H_A + H_B - H_AB
T5   = max(0.0, min(1.0, 1.0 - I_AB / (H_A + 1e-9)))
```

証拠分類：**Derived**（同時観測データがある場合）/**Assumed**（推定の場合）

### 𝓣 ベクトルの扱い方

#### KPI 表示

𝓣 は5成分を個別に表示する。スカラー集約は行わない。

```
KPI カード（𝓣）：
┌────────┬────────┬────────┬────────┬────────┐
│  T₁    │  T₂    │  T₃    │  T₄    │  T₅    │
│スケール │距離    │非対称差 │曲率差  │損失率  │
│差      │        │(α≠0)   │        │        │
├────────┼────────┼────────┼────────┼────────┤
│値・分類 │値・分類 │値・α値 │値・分類 │値・分類 │
└────────┴────────┴────────┴────────┴────────┘
```

#### 介入方向の読み方

| 成分 | 大きい場合の意味 | 介入の方向性 |
|------|----------------|------------|
| $T_1$ 大 | スケール・自由度の乖離 | スケール調整・段階的移行 |
| $T_2$ 大 | 状態点が統計多様体上で遠い | 状態の段階的ブリッジ設計 |
| $T_3$ 大 | 意味変換の非対称差が大きい | コミュニケーション設計・翻訳 |
| $T_4$ 大 | 多様体の整合性が崩れている | 構造的整合性の回復 |
| $T_5$ 大 | エッジの情報損失が大きい | チャネル開放・認知負荷低減 |

#### 推奨計算戦略

| データ状況 | 推奨 |
|-----------|------|
| 両レジームの時系列が豊富 | T₁・T₂・T₃・T₅ を Derived で計算、T₄ を Assumed で補完 |
| データが少ない | T₂（Bhattacharyya）のみ Derived、他は Assumed として 0 |
| 同時観測データなし | T₅ = 0（Assumed）として除外 |

## 解析プロセス（4ステップ）

### Step 1: Input Audit

```
・データ種類・粒度・欠損を確認
・各データに証拠分類ラベルを付ける
・レジームAとBの識別方法を確認
・𝓣 の各成分に使えるデータ量の確認
```

データが存在しない場合の代理推定（全て Hypothesis ラベル）：
```
S → 行動多様性・需要変動係数
D → 導線複雑度・競合多層性
H → トレンド継続期間
F → 人口密度×所得 − 競合密度×S
𝓣 → 定性的推定（各成分 0〜1 のスケール）
```

### Step 2: SDFT Mapping

```
ノード   = 構成要素（部門・ゾーン・セグメント）
リンク   = 接続（強度・方向のみ）
場       = 外部環境（市場・競合・マクロ）
境界     = レジームの境界
レジーム = 同一オペモードを持つ領域
```

リンクは「接続の有無・方向・強度」のみ。四元数・ゲージ歪みは使用しない。

### Step 3: S/D/H/F/𝓣 算出

```python
# 構造場変数（時系列から自動計算）
S = calc_entropy(x)          # Derived
D = higuchi_fd(x)             # Derived
H = hurst_exponent(x)         # Derived
F = U - Theta * S             # Assumed（U・Θが外部入力の場合）

# 𝓣 ベクトル（ハイパーパラメータなし）
T = calc_regime_tension_vector(
    data_A=regime_A,
    data_B=regime_B,
    alpha_T3=1.0   # α≠0 必須（+1 または -1）
)
# T.T1, T.T2, T.T3, T.T4, T.T5 を個別に使用
```

**優先解釈順序：**
1. **dH/dt**（最重要先行予兆）
2. **Var(D)**（構造不安定化）
3. **dS/dt**（遅行指標）
4. **𝓣 の各成分の分布**（どの次元の不整合が大きいか）
5. **F の偏在**（介入レバー）

### Step 4: 位相判定

| Phase | S | D | H |
|-------|---|---|---|
| Living System | ≤0.6 | 1.2〜1.7 | ≥0.45 |
| Frozen Order | ≤0.4 | ≤1.4 | ≥0.55 |
| Runaway Growth | 中 | ≥1.6 | ≥0.6 |
| Noise Dominant | ≥0.65 | ≥1.5 | ≤0.45 |
| Collapse | ≥0.8 | 不安定 | ≤0.3 |

相転移予兆（3条件が全て揃った場合のみ）：

$$\text{alert} = \left(\frac{dH}{dt} < 0\right) \land \left(\mathrm{Var}(D)\text{ 増大}\right) \land \left(\frac{dS}{dt} > 0\right)$$

## 巻末用語集（v0.0.3）

| 用語 | 説明 |
|------|------|
| **S（エントロピー）** | 分布の散逸度。高いほど無秩序 |
| **D（フラクタル次元）** | 時系列の複雑性。1.0〜2.0 |
| **H（Hurst指数）** | 持続性。>0.5 で持続傾向、<0.5 で反転傾向 |
| **F（自由エネルギー）** | 変化の駆動力。$F = U - \Theta S$ |
| **𝓣（レジーム間テンション）** | 5次元ベクトル。2レジーム間の情報幾何学的不整合 |
| **T₁（対数分配関数差）** | スケール・自由度の乖離 |
| **T₂（Bhattacharyya距離）** | 状態点の幾何学的ずれ（Fisher-Rao近似） |
| **T₃（α-ダイバージェンス）** | 非対称な意味的差（α≠0） |
| **T₄（統計的ホロノミー）** | 多様体の曲率・整合性の差 |
| **T₅（通信路損失率）** | エッジ通過時の情報損失割合 |
| **統計多様体** | 確率分布の集合に幾何学的構造を与えた空間 |
| **Fisher情報行列** | 統計多様体上の計量テンソル |
| **対数分配関数** | $\psi(\theta) = \log Z(\theta)$。分布のポテンシャル |
| **Bhattacharyya係数** | $BC = \int\sqrt{p_A \cdot p_B}\,dx$。分布の重なり量 |
| **α-ダイバージェンス** | 情報幾何学における分布間距離の一般化 |
| **統計的ホロノミー** | 統計多様体上の閉ループを一周した際のズレ |
| **通信路損失率** | $\mathcal{L}(e) = 1 - I(X_A;X_B)/H(X_A)$ |
| **相互情報量** | $I(X;Y) = H(X)+H(Y)-H(X,Y)$ |
| **Living System** | 中S・中D・中〜高H。健全な動的平衡 |
| **Frozen Order** | 低S・低D・高H。安定だが変化なし |
| **Runaway Growth** | 中S・高D・高H。過成長の兆候 |
| **Noise Dominant** | 高S・高D・低H。構造崩壊の前兆 |
| **Collapse** | 高S・不安定D・低H。持続不能 |
| **Observed** | 実測値 |
| **Derived** | 観測値から確定的に算出した値 |
| **Assumed** | 分析者の仮定値 |
| **Hypothesis** | 解釈的・推論的推定 |
